Live scanner

Run a live market scan on the latest completed hourly snapshot. The scanner evaluates only live-eligible direct rules and returns a ranked shortlist of matching coins.

Latest snapshot only
Choose live-eligible direct rules on the Rules tab or run all live-eligible direct rules. The scan refreshes the latest bars, computes the latest completed snapshot, evaluates live rules, ranks the shortlist, and writes a downloadable scan pack.
The live scanner is decision support only. It does not execute trades and it excludes non-direct or non-live-eligible rules by default.

Live shadow validation

Evaluate selected direct rules on the latest completed hourly bar, freeze the signal snapshot, and resolve H1/H4/H24 outcomes on later cycles without placing trades.

Read-only
Choose selected direct rules on the Rules tab or run all direct rules. The cycle refreshes live bars, computes the latest feature snapshot, evaluates the latest completed bar, and resolves any matured H1/H4/H24 outcomes from prior signals.
This is shadow validation, not execution. Signals are logged at decision time and compared against realized Coinbase outcomes later.
